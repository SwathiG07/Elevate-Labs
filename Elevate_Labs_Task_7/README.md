Project Overview:
This project demonstrates how to perform basic sales analysis by executing SQL queries inside Python using a SQLite database. An imported CSV dataset was used to calculate sales summaries and visualize results.

Tools Used:
Python, 
SQLite (sqlite3), 
Pandas, 
Matplotlib, 
Jupyter Notebook

Dataset:sales_data.csv

Work Performed:

Imported the CSV dataset into a Jupyter Notebook

Created a SQLite database using Python

Loaded the dataset into the database using to_sql()

Executed SQL queries inside Python to:

Calculate total quantity sold

Generate year-wise sales summary

Displayed query results using print statements

Created and saved a bar chart for year-wise sales quantity using matplotlib

Also Created a new table and performed all the above queries

Output:

SQL query results displayed in the notebook

Bar chart saved as an image file (sales_chart.png)

All operations performed using SQLite within Python

Conclusion:
This task helped in understanding how SQL can be integrated with Python to analyze structured data, generate summaries, and create basic visualizations using a small SQLite database.
