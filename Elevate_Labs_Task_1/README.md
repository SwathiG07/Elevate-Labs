This repository contains a data cleaning and preprocessing task performed on five different real-world datasets using Python and Pandas.

The objective of this task was to identify data quality issues like missing values, duplicates, inconsistent text formats, incorrect data types, and unclean column names, and fix them using standard data preprocessing techniques.

All cleaning steps were implemented in the Jupyter Notebook ElevateLabs_Task_1.ipynb.


Tools & Technologies Used:

Python,
Pandas,
Jupyter,
Notebook

Github


Common Data Cleaning Tasks Applied to All Datasets:

Identified missing values using df.isnull().sum()

Handled missing values using fillna() or dropna()

Removed duplicate rows using drop_duplicates()

Standardized text values (lowercase, removed spaces)

Converted date columns to datetime

Renamed columns to clean and consistent format

Verified and corrected data types using df.info()


Final Outcome:

Performed EDA

All five datasets are now clean, consistent, and analysis-ready

Data quality issues were systematically identified and resolved
