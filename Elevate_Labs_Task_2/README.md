Task 2: Data Visualization and Storytelling

Objective:
To create meaningful visualizations that communicate business insights clearly using Power BI.

Dataset:
Superstore Sales Dataset containing order, sales, profit, customer, and regional information.

Tools Used:
- Power BI Desktop

Process:
1. Cleaned and prepared data in Power Query
2. Created DAX measures for Sales, Profit, and KPIs
3. Designed visuals based on data type and business questions
4. Reduced clutter and maintained consistent color usage
5. Added context and insight-driven titles to each chart
6. Created a final storyboard to summarize key insights

Key Insights:
- Sales grew steadily, but profit margins were inconsistent
- Technology category is the most profitable
- High discounts significantly reduce profitability
- Certain regions perform poorly despite high sales

Outcome:
This project demonstrates the use of data visualization as a storytelling tool to support data-driven business decisions.
