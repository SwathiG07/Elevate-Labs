Objective:

The objective of this project is to analyze order volume and revenue trends from an online store dataset using SQL. 

Tools Used:

DB Browser for SQLite

Dataset

The dataset used for this project is Online Store Order data.

Files Submitted:

The following files have been uploaded as part of this project:

--Dataset File

--SQL Script File

    Includes all SQL queries written for:

      Date extraction (year and month)

      Revenue calculations

      Order volume analysis

      Grouping and aggregation

      Trend and growth analysis

--Results Table File (DOC)

Execution Environment:

All SQL queries were executed and tested using DB Browser for SQLite, and results were captured directly from the tool.

Outcome:

This project demonstrates the use of SQL aggregation, grouping, and date functions to analyze time-based trends and derive meaningful business insights from transactional data.
